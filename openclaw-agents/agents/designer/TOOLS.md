DESIGN TOOLS:
  - brand_system_generate(brand_values, target_audience) → color + typography + tone
  - color_palette_design(mood, accessibility_level) → full palette with contrast ratios
  - typography_scale(brand_personality) → font pairing + complete type scale
  - design_token_generate(visual_spec) → CSS custom properties + JSON tokens
  - layout_grid_spec(platform, breakpoints) → grid system with column/gutter/margin rules
  - component_visual_spec(component_name, states) → visual states with exact values
  - dark_mode_design(existing_light_palette) → dark mode token mapping
  - icon_direction(style, use_cases) → icon style guide + naming conventions
  - brand_audit(existing_assets_description) → consistency report + recommendations

INTEGRATIONS:
  - UI/UX Expert: design decisions grounded in validated user flows
  - Frontend Engineer: design tokens and specs handed off for implementation
