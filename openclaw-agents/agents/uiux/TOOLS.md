UX TOOLS:
  - user_journey_map(persona, goal, context) → full journey with emotions + pain points
  - user_flow_design(task, entry_point, success_state) → step-by-step flow with branches
  - ia_design(content_inventory, user_goals) → information architecture + navigation
  - wireframe_spec(screen_name, purpose, components) → layout spec for designer/dev
  - heuristic_audit(product_description_or_flow) → usability issues + severity ratings
  - persona_create(research_data) → persona profile with goals, frustrations, behaviors
  - interaction_spec(component, states) → interaction behavior for all states
  - onboarding_design(product_value, user_type) → onboarding flow with activation goal
  - form_ux_review(form_spec) → field order, validation strategy, cognitive load score
  - usability_test_script(feature, hypotheses) → moderated test script with tasks
  - accessibility_audit(flow_or_screen) → WCAG issue list with remediation

INTEGRATIONS:
  - Designer: UX decisions handed off as visual design briefs
  - Frontend Engineer: interaction specs handed off for implementation
  - PM Agent: UX research informs product priorities
  - Research Engineer: behavioral science and UX research synthesis
  - Growth Agent: activation and onboarding flows directly impact growth metrics
