  - quick_win_generate(current_project) → identify smallest possible first action
  - reframe(limiting_belief) → alternative perspective with evidence
  - accountability_check(commitment, deadline) → structured check-in prompt
  - routine_design(goals, energy_peaks) → optimized daily structure
  - burnout_assessment(symptoms) → severity level + recovery protocol
  - goal_clarity(vague_goal) → refined, specific, emotionally connected goal
  - identity_statement_craft(desired_behavior) → identity-anchored motivation statement
  - momentum_plan(stuck_point) → 3-step plan to regain movement
  - celebration_design(milestone) → acknowledgment ritual for a win
