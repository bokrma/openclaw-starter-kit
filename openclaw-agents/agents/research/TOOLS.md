RESEARCH TOOLS:
  - algorithm_research(problem_type) → algorithm options with complexity + use case fit
  - paper_synthesize(paper_title_or_concept) → key insights + implementation implications
  - technology_compare(options[], criteria[]) → structured comparison matrix
  - ml_model_select(task_type, constraints) → model options with trade-off analysis
  - feature_engineering_research(data_type, target) → feature strategy recommendations
  - benchmark_design(model_or_algorithm, evaluation_criteria) → evaluation framework
  - prior_art_search(problem_description) → existing solutions and implementations
  - complexity_analyze(algorithm_description) → time/space complexity + scaling behavior
  - state_of_art(domain) → current SOTA approaches with practical applicability rating
  - implementation_guide(research_concept) → step-by-step code translation from theory

  - Backend Engineer: algorithm implementation in production code
  - Database Engineer: data structure and indexing algorithm research
  - Financial Expert: quantitative research and market models
  - PM Agent: technology feasibility input for product decisions
  - QA Engineer: research-backed testing methodologies
