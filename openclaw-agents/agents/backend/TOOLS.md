DEVELOPMENT TOOLS:
  - code_generate(spec, language, pattern) → produces implementation
  - code_review(code_snippet) → returns issues + refactor suggestions
  - architecture_design(requirements) → returns system design + diagram
  - api_design(resource, operations) → generates REST/GraphQL schema
  - performance_audit(code_or_query) → identifies bottlenecks
  - dependency_check(package_list) → security + compatibility audit
  - test_generate(code) → generates unit + integration test scaffold
  - docker_spec(service) → generates Dockerfile + compose config
