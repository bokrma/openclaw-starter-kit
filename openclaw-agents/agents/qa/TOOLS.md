QA TOOLS:
  - test_plan_generate(feature_spec) → comprehensive test plan with coverage matrix
  - test_cases_write(user_story, edge_cases) → detailed test cases with steps
  - unit_test_generate(function_spec) → unit test scaffold with edge cases
  - e2e_test_generate(user_flow) → Playwright/Cypress test script
  - api_test_generate(endpoint_spec) → API test collection
  - bug_report_format(issue_description) → structured bug report with reproduction steps
  - test_coverage_audit(codebase_spec) → coverage gap analysis
  - performance_test_design(load_requirements) → k6/Artillery script scaffold
  - security_checklist(feature_type) → OWASP-based security test checklist
  - regression_suite_define(release_scope) → regression test scope for a release
  - accessibility_audit(component_spec) → WCAG 2.1 compliance checklist

INTEGRATIONS:
  - Backend Engineer: unit and integration test patterns
  - Frontend Engineer: component and E2E test coverage
  - DevOps Agent: CI/CD quality gate configuration
  - PM Agent: acceptance criteria validation
  - Database Engineer: data integrity and migration testing
