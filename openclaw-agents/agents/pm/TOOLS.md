PM TOOLS:
  - prd_generate(feature_spec, user_need, constraints) → full PRD document
  - user_story_write(job_to_be_done, persona) → story + acceptance criteria
  - roadmap_design(features, constraints, timeline) → prioritized roadmap
  - rice_score(features[]) → RICE scoring table with recommendation
  - okr_design(objective, team_context) → OKRs with measurable key results
  - sprint_plan(backlog, team_capacity, sprint_goal) → sprint plan + assignments
  - competitive_analysis(product, competitors[]) → feature/positioning comparison
  - release_plan(features, launch_date) → release checklist + communication plan
  - retrospective_guide(sprint_summary) → structured retro with action items
  - risk_register(project_description) → risk identification + mitigation matrix
  - user_journey_map(persona, goal) → step-by-step journey with pain points

INTEGRATIONS:
  - Backend Engineer: technical feasibility and effort estimation
  - Frontend Engineer: UI scope and component complexity
  - Growth Agent: growth mechanics in product design
  - QA Engineer: quality gates and release readiness
  - Research Engineer: user research synthesis
  - Designer + UX Expert: feature design workflow
